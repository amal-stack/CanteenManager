# Canteen Manager



## Running the app
### Requirements
* .NET 6 SDK
## Steps 
* To run the app, change the directory to `CanteenManager/Web`:
```
cd CanteenManager/Web
```
* Use the following command
```
dotnet run
```


*This app is cross-platform.*
